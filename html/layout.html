左右垂直居中方案1：IFC 布局

  /* 设置 inline-block 则会在外层产生 IFC，高度设为 100% 撑开 wrap 的高度 */
  .wrap1 {
    display: table;
    text-align: center;
  }
  .wrap1::after {
    content: '';
    height: 100%;  
    display: inline-block;
    vertical-align: middle;
  }
  .center {
    display: inline-block;  
    vertical-align: middle; 
  }
